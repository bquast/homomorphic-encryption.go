WASM CKKS Encryption

Approximate Homomorphic Encryption for Complex Vectors.

← Back to Schemes

Pipeline Stages

1. Vector Encoding

Converts complex vector z into an integer polynomial.

2. Setup Phase (RLWE)

Generate Public Key (pk) and Secret Key (sk) for the Ring structure.

3. Encryption Phase

Masks the integer polynomial m(X) into ciphertexts (c0, c1).

4. Decryption Phase

Uses the Secret Key to recover the noisy polynomial m'(X).

5. Vector Decoding

Evaluates and un-scales m'(X) back to the approximate vector z'.
